# Doctor Appointment System (MERN)

## Features
- User authentication (JWT)
- Book appointments
- Admin dashboard
- Role-based access

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## Folder Structure
- frontend/
- backend/
- admin/

## Run Locally

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start

### Admin
cd admin
npm install
npm start
